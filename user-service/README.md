# User Service

Сервис управления профилями пользователей.

## Функционал

- Создание профиля при регистрации (через Kafka события от Auth Service)
- Получение профиля пользователя
- Обновление имени, аватара, настроек
- Управление статусом (online/offline) через Redis

## Архитектура

- **PostgreSQL** - хранение профилей
- **Redis** - кэширование статусов
- **Kafka** - общение с API Gateway и другими сервисами

## Запуск

```bash
docker-compose up -d
go run cmd/user-service/main.go
```
